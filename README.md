# Code Agent CLI Statuslines

A collection of statusline configurations and helper scripts for AI coding agent CLIs on Windows.

## Supported CLIs

* **[Claude Code](./claude-code/README.md)**: A PowerShell-based statusline showing the active model, effort, workspace folder, Git status, and rate limit usage.
* **[Antigravity CLI](./antigravity-cli/README.md)**: A Python-based statusline showcasing the model, folder, Git information, context usage, and API quotas (Gemini & Claude).

## Installation

Please navigate to the directory of the CLI you want to configure and follow the setup instructions in its respective README:
- [Claude Code Statusline Setup](./claude-code/README.md#installation)
- [Antigravity CLI Statusline Setup](./antigravity-cli/README.md#installation)
