# Code Agent CLI Statuslines

A collection of statusline configurations and helper scripts for AI coding agent CLIs on Windows.

## Supported CLIs

* **[Claude Code](file:///C:/Users/hjc/WorkSpace/code-agent-cli-statusline/claude-code/README.md)**: A PowerShell-based statusline showing the active model, effort, workspace folder, Git status, and rate limit usage.
* **[Antigravity CLI](file:///C:/Users/hjc/WorkSpace/code-agent-cli-statusline/antigravity-cli/README.md)**: A Python-based statusline showcasing the model, folder, Git information, context usage, and API quotas (Gemini & Claude).

## Installation

Please navigate to the directory of the CLI you want to configure and follow the setup instructions in its respective README:
